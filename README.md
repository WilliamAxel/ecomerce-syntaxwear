## SyntaxWear - E-commerce de Tênis e Sneakers

Projeto de uma landing page para loja virtual de tênis e sneakers, com layout moderno, responsivo e organizado, utilizando apenas HTML5 e CSS3.

### Funcionalidades
- Navegação por categorias (Masculino, Feminino, Outlet)
- Seção hero com destaque de produto
- Grid de produtos em destaque
- Formulário de newsletter
- Rodapé com links úteis e redes sociais
